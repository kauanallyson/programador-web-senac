A Linguagem de programação Java

Quando surgiu?

Nasceu em 1995 na Sun Microsystems, originalmente com o nome de Oak. Seu principal responsável foi James Gosling. Em 2009, a Sun foi adquirida pela Oracle por cerca de 7,4 bilhões de dólares. Ganhando assim os direitos do Java e de outros produtos, como o MySQL (Banco de dados relacionais).

Por que do nome Java?

Vem de um café que os criadores da linguagem adoravam tomar, o café da Ilha de Java na indonésia. Desse mesmo café vem também a icônico logo da linguagem!

Paradigma

Java é totalmente orientada à objetos, com grande inspiração no SmallTalk dos anos 80. Tudo é um objeto, com propriedades e métodos próprios

Compilado ou interpretado?

Um pouco dos dois... Os arquivos .java são compilados para arquivos .class que podem ser interpretados pela JVM.

JVM?

JVM - Java Virtual Machine é o que dá sentido ao slogan do Java: "Write once, run anywhere" - "Escreva uma vez, rode em qualquer lugar" em tradução livre. Se trata de um programa que executa o seu código, sem precisar ser compilado para cada sistema diferente. Um programa escrito para Mac funciona no Windows sem adaptações. Esse é o poder da JVM.

Onde é usado?

Por muito tempo Java foi o líder de mercado para desenvolvimento web, mobile, sistemas e etc. Graças a JVM o ele dominou inúmeras plataformas por muitos anos. Sua principal aplicação nos dias de hoje é Programação Web Backend com o Framework SpringBoot, em Sistemas Desktop legado e grandes jogos como Minecraft e o recém lançado Hytale

Boa para iniciantes?

Para entender sobre orientação à objetos Java é uma excelente linguagem. Ela te leva a pensar em tudo no seu código como uma classe. Para uso comercial é ainda melhor, sua robustez, ecossistema e portabilidade fazem a linguagem ser a queridinha das grandes empresas financeiras e sistemas distribuídos

Hello World

            
public static void main(String[] args){ 
  System.out.println("Hello World");
}

Wow, isso realmente parece bem grande. Mas vou te mostrar que é mais fácil de entender do que parece...

public static void main(String[] args)

esse é o ponto de entrada do nosso código, estamos dizendo ao Java que queremos executar todo o código que está entre chaves. Simples assim.

System.out.println("Hello World");

Ok... Comparado com linguagens como python, o Java é bem extenso no seu "print". Vamos por partes. System é uma classe utilitária que contém vários métodos que interagem diretamente com o sistema. out é uma propriedade especifica, os dados de saída do programa. println é o nosso método (ou função) de ouro! Ele imprime no terminal tudo que está entre as aspas, no caso acima a String "Hello World". Por fim, o ; no final avisa ao Java que terminamos de escrever uma linha de código.

Pronto! Viu como foi simples? Muitos desenvolvedores se afastam da linguagem logo no primeiro momento por conta dessa idéia que precisa escrever muitas linhas de código...

Considerações finais

Alguns programadores possuem receio ao trabalhar com a linguagem, pensam que é um dinossauro corporativo e lento. Mas a realidade conta outra história. A linguagem superou a provação do tempo e até hoje continua nos topo dos rankings de linguagens. A orientação à objetos pode assustar um pouco, mas ela contribui com o desenvolvimento de grandes aplicações abstraindo grande parte da lógica deixando você focar na regra de negócio.
